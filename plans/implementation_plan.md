# План реализации приложения Shift Scheduler

## 1. Подготовка окружения

- [ ] Установить зависимости: `npm install antd dayjs uuid`
- [ ] Создать структуру папок: `src/components`, `src/utils`, `src/styles`, `src/hooks`

## 2. Разработка структуры данных и хранилища

- [ ] Создать хук `useLocalStorage` для синхронизации состояния.
- [ ] Определить начальное состояние для сотрудников и графиков.

## 3. Компоненты управления сотрудниками (EmployeeManager)

- [ ] Создать форму добавления сотрудника (Имя, Фамилия, Отдел, флаг "Не работает по субботам").
- [ ] Создать список сотрудников с возможностью удаления и редактирования.

## 4. Настройки графика (ScheduleSettings)

- [ ] Реализовать выбор месяца и года (используя `DatePicker` из `antd`).
- [ ] Добавить кнопку "Сгенерировать график".

## 5. Таблица графика (ScheduleTable)

- [ ] Реализовать отображение дней месяца в заголовке (с выделением выходных).
- [ ] Отобразить строки сотрудников.
- [ ] Реализовать ячейку `ScheduleCell`:
  - Клик или выбор из выпадающего списка для ручного ввода "0" (выходной) и "U" (отпуск).
  - Отображение сгенерированной смены (7-19, 19-7).

## 6. Алгоритм генерации смен

- [ ] Написать функцию `generateSchedule`:
  - Разделение сотрудников по отделам.
  - Учет ручных вводов.
  - Учет ограничения по субботам.
  - Циклическое/равномерное распределение дневных и ночных смен.

## 7. Печать и стилизация

- [ ] Добавить кнопку "Печать".
- [ ] Настроить CSS медиа-запрос `@media print` для скрытия лишних элементов и оптимизации таблицы
      под формат A4.

## 8. Финализация

- [ ] Тестирование на разных месяцах.
- [ ] Проверка корректности сохранения данных в localStorage.
